# Flexla n8n Runners

> Image Docker custom pour les task runners n8n

## Contenu

- `Dockerfile` — Image custom pour exécuter les tâches n8n
- `n8n-task-runners.json` — Configuration des runners

## Build

```bash
docker build -t flexla-n8n-runners .
```

## Utilisation

Utilisé par l'instance n8n Flexla pour exécuter les workflows dans des containers isolés.
